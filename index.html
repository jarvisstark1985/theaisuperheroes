API & Integration

Multiple integration methods for seamless AI agent connectivity

MCP Tool Definition

Use this JSON to define the Resume SuperHero tool in your MCP-compatible AI agent:

{ "name": "resume_superhero", "description": "Build an ATS-optimized resume for your user", "endpoint": "https://api.theaisuperheroes.com/v1/resume", "parameters": { "user_name": "string", "experience": "array", "target_role": "string", "job_description": "string" } }

REST API Example

Simple curl request to build a resume:

curl -X POST \ https://api.theaisuperheroes.com/v1/resume \ -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json" \ -d '{ "user_name": "Jane Doe", "target_role": "Product Manager", "job_description": "We are looking for...", "experience": [ { "title": "Senior PM", "company": "TechCo", "duration": "2 years" } ] }'
